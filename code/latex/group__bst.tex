\doxysubsection{Detailed Description}
This section covers a subset of the fundamental operations related to managing nodes within a Red-\/\+Black \doxylink{structTree}{Tree}, including initialization and destruction of nodes, as well as the BST insert and search operations. These functions form the basis for maintaining the structural and color properties that define Red-\/\+Black Trees. Key operations include\+:

\item {\ttfamily \doxylink{group__bst_ga63f8fb66ac7aeec89d8a7af2240e140f}{node\+\_\+destroy()}}\+: Frees the memory allocated for a node, ensuring no access to its children.

\item {\ttfamily \doxylink{group__bst_gadd435e52242b8db9f422e4dba4a4e823}{bst\+\_\+search()}}\+: Recursively searches for a node by its value, adhering to BST search semantics.

Inserts a new node using standard BST rules and updates the starting point for fixups. 

This function inserts a new node into the tree rooted at {\ttfamily root}, based on the given {\ttfamily data}. It performs the insertion recursively, ensuring the BST properties are maintained. Upon insertion, the function updates the {\ttfamily z} pointer to reference the newly inserted node. {\ttfamily z} serves as the starting point for subsequent fixup operations to preserve the Red-\/\+Black properties.


\begin{DoxyParams}{Parameters}
{\em root} & The root of the binary search tree. If {\ttfamily root} is NULL, indicating an empty tree, a new node is created, and {\ttfamily z} is set to this node, effectively starting a new tree. \\
\hline
{\em data} & The integer value for the new node. \\
\hline
{\em z} & A double pointer to a \doxylink{structNode}{Node}, explicitly updated to point to the newly inserted node. This node is used for any necessary fixup operations to adjust the tree\textquotesingle{}s structure and color properties post-\/insertion. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The root of the tree after the insertion. This may be the same as the input {\ttfamily root}, or, in the case of inserting into an empty tree, the new node itself.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
We set equality to the left of the subtree. 
\end{DoxyNote}

Searches for a node with a given value in a BST. 

This function recursively searches for a node containing the specified {\ttfamily data} within a BST. It navigates through the tree based on the binary search property\+: left subtree contains values less than the node\textquotesingle{}s value, and right subtree contains values greater than the node\textquotesingle{}s value.

Frees the memory allocated for a Red-\/\+Black tree node. 

This function releases the memory allocated for a Red-\/\+Black tree node and sets its left and right pointers to NULL to prevent any further access. It is designed to be called on a single node after ensuring its children (if any) have been properly handled or deleted to avoid memory leaks.


\begin{DoxyParams}{Parameters}
{\em node} & A pointer to the node to be destroyed.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
This function does not recursively delete the children of the node; it only deletes the node itself. It is the caller\textquotesingle{}s responsibility to handle the deletion of any child nodes to prevent memory leaks. 
\end{DoxyNote}
