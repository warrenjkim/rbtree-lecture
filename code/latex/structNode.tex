\doxysubsection{Detailed Description}
\doxylink{structNode}{Node} structure for a Red-\/\+Black \doxylink{structTree}{Tree}. 

Each node in the Red-\/\+Black \doxylink{structTree}{Tree} contains the data it stores, pointers to its left and right children, a pointer to its parent, and its color.
